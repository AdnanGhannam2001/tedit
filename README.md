# TEdit

A simple text editor built with `C++17` using [`SFML`](https://www.sfml-dev.org/) only

## Keybindings

- `C-f`: Go forward
- `C-b`: Go backward
- `C-n`: Next line
- `C-p`: Previous line
- `C-a`: Go to then beginning of line
- `C-e`: Go to the end of line
- `C-d`: Delete forward
- `C-h`: Delete backward
- `C-space`: Switch select mode
- `C-x`: Cut selected
- `C-w`: Copy selected
- `C-y`: Paste
- `C-s`: Save
- `C-o`: Open
